## Как использовать этот чек-лист

- [ ] = Не проверено
- [x] = Проверено, работает
- [!] = Найден баг (опишите в комментариях)

**Для каждого бага укажите:**
1. Шаги воспроизведения
2. Ожидаемый результат
3. Фактический результат
4. Скриншот (если возможно)
